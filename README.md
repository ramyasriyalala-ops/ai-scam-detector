# AI Scam Detector

## Problem Statement
Many people in Tier-2 and Tier-3 cities fall victim to fake job and loan scams.

## Solution
AI-powered scam detection system using NLP to detect fake job and loan messages.

## Features
- Detect fake job offers
- Detect fake loan messages
- NLP-based text analysis
- User-friendly interface

## Tech Stack
- Python
- NLP
- Machine Learning
- AWS (optional)

## Future Scope
- WhatsApp integration
- Regional language support
- Mobile application
